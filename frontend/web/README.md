# Frontend estático: `frontend/web`

Arquivos simples de demonstração com a landing page e tela de login.

Como usar:

1. Rodar o frontend Next.js (pasta `frontend/web`):

```bash
cd frontend/web
npm install
npm run dev
```

Isso iniciará o site em `http://localhost:3001`.

Observação: as páginas em `pages/` são templates Next.js; remova ou adapte os antigos arquivos estáticos se desejar.
